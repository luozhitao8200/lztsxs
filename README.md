# 实训室管理系统

一个基于 Web 的实训室预约和管理系统。

## 功能特点

- 实训室状态实时显示（空闲、预约、使用中）
- 设备管理
- 预约管理
- 使用情况汇总
- 管理员权限控制

2. 管理员功能
   - 添加/编辑/删除实训室需要管理员密码：admin

3. 预约功能

   - 填写预约信息
   - 系统自动更新状态
